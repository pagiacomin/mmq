The Project of Multi-functional Missions Queue

This page is a complement to the paper ‘A distributed, real-time and easy-to-extend strategy for missions of autonomous aircraft squadrons’, which was published in the Information Sciences magazine in 2022, and in which the problem of executing the multifunctional missions queue is proposed and solved. Basically, this page documents the main procedures and data structures used in the simulation’s source code, see https://pagiacomin.github.io/mmq/.
